# 🏠 Dream Homes - Real Estate Website

A modern and responsive real estate listing website built using **HTML, CSS, and JavaScript**. This project showcases property listings with search and filtering features, along with an attractive UI and smooth user experience.

---

## 🚀 Features

- 🌟 Fully responsive design (mobile + desktop)
- 🎞️ Hero section with auto-sliding background images
- 🔍 Search functionality (filter by city/type)
- 💰 Price filter (low, mid, high range)
- 🌙 Dark mode toggle
- 🧊 Modern UI with clean layout
- 🏠 Property cards with images, price, and location
- 📩 Contact form section
- ⚡ Smooth animations and hover effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (DOM Manipulation)
